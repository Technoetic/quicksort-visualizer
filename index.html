퀵 정렬이란?

퀵 정렬은 피벗(pivot)을 선택하여 배열을 두 부분으로 분할하고, 각 부분을 재귀적으로 정렬하는 알고리즘입니다.

평균 시간 복잡도 O(n log n)으로, 실제 환경에서 매우 빠른 성능을 보입니다.

  • 분할 정복(Divide & Conquer) 패러다임
  • 제자리(in-place) 정렬 알고리즘
  • 불안정(unstable) 정렬
  • 재귀 구조로 구현

피벗 선택의 중요성

피벗을 중앙값에 가까운 원소로 선택할수록 분할이 균등해져 최적 성능에 가까워집니다.

  1. function quickSort(arr, low, high)
  2. if low < high then
  3. pivot ← arr[low]
  4. p ← partition(arr, low, high)
  5. quickSort(arr, low, p-1)
  6. quickSort(arr, p+1, high)
  7. function partition(arr, low, high)
  8. pivot ← arr[low]; i ← low
  9. for j ← low+1 to high do
  10. if arr[j] ≤ pivot then
  11. i++; swap(arr[i], arr[j])
  12. swap(arr[low], arr[i])
  13. return i
경우복잡도조건
최선O(n log n)균등 분할
평균O(n log n)일반적
최악O(n²)이미 정렬됨

공간 복잡도: O(log n) (재귀 스택)
